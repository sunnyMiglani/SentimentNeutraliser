This page will tell you about Sentimentally!

Sentimentally is an autocorrect tool focused on sentiment!
Given a sentence with some sentiment value in it, the algorithm and system being applied will find and replace these words with alternative words that change the overall sentiment of the sentence.

You can see some examples of this here!.

The goal with this project in general is

  • Allow users to pick a sentiment of their choosing (+ve, -ve, neutral)
  • Suggest alterations in the words where the alterations hopefully match the Parts-Of-Speech of the sentence
  • Currently it only changes words, but does not add or remove any words
  • Currently the program gives you all possible alterations, not allowing users to pick a sentiment value!
